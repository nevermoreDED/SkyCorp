# SkyCorp
Все обновления приложения корпорации SkyCorp

Этот репозиторий содержит установочные файлы для приложения SkyCorp.

## Как обновить приложение
1. При появлении уведомления об обновлении нажмите "ОБНОВИТЬ СЕЙЧАС"
2. Браузер откроет страницу загрузки
3. Скачайте архив с последней версией и запустите установщик
4. Следуйте инструкциям установщика
